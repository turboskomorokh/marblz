# marblz

Моя практика в OpenGL и создании игрового движка.

Передвижение: WASD + SPACE/SHIFT + Мышь  
Переключение камеры: 1

```shell
$ sudo apt update && sudo apt install -y \
  libgl1-mesa-dev \
  libglfw3-dev \
  libglew-dev \
  libglm-dev \
  libpng-dev \
  libspdlog-dev

```
